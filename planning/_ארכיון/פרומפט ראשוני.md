# פרומפט ראשוני

> [!abstract] הוראות שנמסרו לקלוד קוד בתחילת הפרויקט

אני רוצה לבנות כלי Python למיגרציית נתונים לסיילספורס. צירפתי קובץ תכנון מפורט.

לפני שמתחילים לכתוב קוד:

1. קרא את קובץ התכנון
2. היכנס ל-plan mode
3. הצע שיפורים, שאל שאלות, וציין אם יש משהו שלא ברור או שכדאי לשנות
4. צור את מבנה התיקיות המוגדר במסמך כולל `.gitignore` עם `credentials.json`

אל תתחיל לכתוב קוד לפני שנסכים על התכנון הסופי.

ואני חוזר שוב — שאל שאלות, הצע הצעות, אל תיקח את התוכנית כמשהו סגור. זו תוכנית שבניתי, אבל הכל פתוח לשינויים. אני גם חדש בתחום — כך שבהחלט ייתכן ששכחתי דברים.
